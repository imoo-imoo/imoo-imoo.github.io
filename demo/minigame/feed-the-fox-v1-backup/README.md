# Feed the Fox

A portrait HTML mini game designed for a 1080 x 1920 display, with responsive scaling for phones and exhibition screens.

## Run

Open `index.html` directly in a browser, or serve this folder locally:

```sh
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173/
```

Scores are saved in the browser on the same device.
